Thoughts on Progressive Enhancement

Progressive enhancement is the practice of basic user experience in my opinion making sure that across devices and browsers the stability is prominent. In web development we ensure that all devices are able to have the same experience versus one browser only having the supported features. User experience is gold in this field of modern web development.
